# Netflix_Pandas_Project
Overview

This project performs data analysis on a Netflix dataset using Python and Pandas. The goal is to explore trends, clean data, and extract meaningful insights about movies and TV shows available on Netflix.

Features
Data cleaning and preprocessing using Pandas
Analysis of movies vs TV shows distribution
Exploration of content by country, genre, and release year
Identification of most frequent directors and actors
Duration and rating analysis
Tech Stack
Python
Pandas
NumPy
Matplotlib / Seaborn (for visualization)
Project Structure
Netflix_Pandas_Project/
│── dataset/
│   └── netflix_titles.csv
│── notebook/
│   └── analysis.ipynb
│── outputs/
│   └── charts/
│── README.md
Key Insights
Majority of Netflix content consists of movies rather than TV shows
The United States has the highest number of titles
Content production has increased significantly after 2015
Drama and International genres dominate the platform
How to Run
Clone the repository

Install required libraries:

pip install pandas numpy matplotlib seaborn

Run the Jupyter Notebook:

jupyter notebook
Conclusion

This project demonstrates how Pandas can be used effectively for real-world data analysis and visualization, providing useful insights into Netflix content trends.
